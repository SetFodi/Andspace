# Security Policy

AndSpace is currently an early macOS alpha. The current public release is
`v0.1.0-alpha.5`.

## Supported Versions

Only the latest public alpha is expected to receive fixes.

| Version | Supported |
| --- | --- |
| `v0.1.0-alpha.5` | Yes |
| Older alpha builds | No |

## Reporting A Vulnerability

For non-sensitive issues, use GitHub issues:
https://github.com/SetFodi/Andspace/issues

For sensitive security concerns, do not post secrets, tokens, private project
data, or working exploit details publicly. Open a minimal issue asking for a
private reporting path, or use GitHub private vulnerability reporting if it is
available on the repository.
