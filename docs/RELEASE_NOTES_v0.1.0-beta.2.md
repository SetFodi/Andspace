# AndSpace v0.1.0-beta.2 Release Notes

AndSpace `v0.1.0-beta.2` is a focused beta update that makes the current app
state public: Agents cockpit, terminal search, tab agent badges, and the latest
daily-use polish are included in the downloadable DMG and ZIP.

- Website: https://andspace.app
- GitHub release: https://github.com/SetFodi/Andspace/releases/tag/v0.1.0-beta.2
- Demo video: https://andspace.app/andspace.mp4
- Privacy: https://andspace.app/privacy
- Security: https://andspace.app/security
- Feedback: https://github.com/SetFodi/Andspace/issues

## What's New Since beta.1

- **Agents cockpit (`Cmd+Shift+A`).** A lightweight overlay for local Claude,
  Codex, and Cursor CLI handoff sessions launched from AndSpace. It shows
  running, completed, and failed sessions and lets you jump back to the related
  terminal pane.
- **Agent-aware tab labels.** Tabs now surface active local AI CLI sessions so
  it is easier to see where an agent is running.
- **Terminal search and copy/paste polish.** The beta.1 terminal find/copy/paste
  work remains included in this release.
- **Latest docs and install wording.** Public README, verification docs, and
  website download flow are aligned to the current beta.

No telemetry, accounts, provider API integrations, Git write actions, or
automatic updates were added. AI handoff continues to use only your locally
installed CLIs.

## Installation

1. Download the macOS DMG or ZIP from the
   [GitHub release](https://github.com/SetFodi/Andspace/releases/tag/v0.1.0-beta.2).
2. If you downloaded the DMG, open it and drag `AndSpace.app` to Applications.
3. If you downloaded the ZIP, unzip it and move `AndSpace.app` to
   `/Applications`.
4. Launch AndSpace.

AndSpace is an unsigned prerelease beta. On first launch macOS may block it.
Open **System Settings -> Privacy & Security**, scroll to the Security section,
and click **Open Anyway**, then confirm once more.
