# AndSpace v0.1.0-alpha.2 Release Notes

AndSpace is a terminal-first macOS app for local development. It is built
around real terminal tabs and panes, with lightweight workflow helpers for
project context, local AI CLI handoff, server discovery, and read-only Git
inspection.

- Website: https://andspace.app
- GitHub release: https://github.com/SetFodi/Andspace/releases/tag/v0.1.0-alpha.2

## Included In This Alpha

- Terminal tabs and split panes.
- Pane focus navigation with `Cmd+Arrow`.
- Workspace restore for tabs, splits, cwd, sidebar state, and window shape.
- Command Guard with project rules from `ANDSPACE.md`.
- `ANDSPACE.md` initializer.
- AI CLI handoff to installed local Claude Code, Codex, and Cursor CLIs.
- Command palette.
- Keyboard shortcuts overlay.
- Optional project sidebar.
- Files, Scripts, and Servers sidebar sections.
- File Actions for Cursor, VS Code, Neovim split, copy path, and Finder reveal.
- Passive localhost detection from terminal output.
- Read-only Git Changes.
- Read-only Git Diff Preview.

## Platform Notes

- macOS-first alpha.
- zsh-first shell integration.
- AI handoff uses local CLIs only.
- No provider API integration, hosted AI backend, API key management, or API
  billing.

## Installation

1. Download the macOS alpha artifact from the
   [GitHub release](https://github.com/SetFodi/Andspace/releases/tag/v0.1.0-alpha.2).
2. Unzip it if the artifact is a `.zip`.
3. Move `AndSpace.app` to `/Applications` if desired.
4. Launch AndSpace.

Because this alpha is not notarized yet, macOS may block the first launch. Use
right-click -> Open from Finder if needed.

## Known Limitations

- Not signed with a Developer ID.
- Not notarized.
- No auto-update.
- macOS-first and zsh-first.
- No settings UI yet.
- No embedded browser preview yet.
- No Git write actions: no staging, commit, push, pull, reset, checkout, stash,
  merge, or rebase UI.
- No file editing or in-app code editor.
- Server detection is passive and best-effort from terminal output only. It
  does not scan ports, poll, ping, or health-check servers.
- Git Diff Preview is read-only and capped for large diffs.

## Notes For Testers

This alpha is intended for local development dogfooding. Please report issues
with terminal focus, pane navigation, workspace restore, Command Guard false
positives, local CLI handoff, Git diff preview, and first-launch packaging.
